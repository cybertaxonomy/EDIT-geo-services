Constructor Detail

ModelBase

ModelBase(modelNode, parentModel)
    Base Model class to be inherited by all Model objects and provdes methods and properties common to all models. Stores the XML document as the .doc property of the model. Inherits from the Listener class so all models are also listener objects that can call registered listeners.
    Parameters:
      modelNode - The model's XML object node from the configuration document.
      parentModel - The model object that this model belongs to.
    Author: Cameron Shorter

loadModelDoc

void loadModelDoc(objRef)
    Load a Model's document. This will only occur if the model.url property is set. Calling this method triggers several events: modelStatus - to indicate that the model state is changing newModel - to give widgets a chance to clear themselves before the doc is loaded loadModel - to indicate that the document is loaded successfully
    Parameters:
      objRef - Pointer to the model object being loaded.

newRequest

void newRequest(objRef, httpPayload)
    Load XML for a model from an httpPayload object. This will also handle instantiating template models if they have the "template" attribute set. To update model data, use:
    httpPayload=new Object();
    httpPayload.url="url" or null. If set to null, all dependant widgets will be removed from the display.
    httpPayload.httpMethod="post" or "get"
    httpPayload.postData=XML or null
    Parameters:
      objRef - Pointer to the model object being loaded.
      httpPayload - an object to fully specify the request to be made
